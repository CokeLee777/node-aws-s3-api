> About Aws s3 put, get, delete static file api by using node.js

## Prerequisites

- npm >= 8.13.2
- node >= 16.5.0

## install

```sh
npm install
```

## Usage

```sh
npm run start
```

- Call api

```
GET: http://localhost:3001/api/s3/buckets/file
POST: http://localhost:3001/api/s3/buckets/file
DELETE: http://localhost:3001/api/s3/buckets/file
```
